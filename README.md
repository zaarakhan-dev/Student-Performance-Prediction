# Student-Performance-Prediction
### Internship Capstone Project

Developed as part of a Data Science Internship Program.

## Project Overview
This project predicts student mathematics scores using machine learning techniques.

## Dataset
Students Performance in Exams Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Methodology
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Encoding
4. Model Training
5. Model Evaluation

## Model Used
Linear Regression

## Results
- Mean Squared Error (MSE): 28.28
- R² Score: 88.4%

## Key Findings
- Reading and writing scores strongly influence mathematics performance.
- Students completing test preparation courses generally perform better.
- Parental education positively affects academic achievement.

## Conclusion
The model achieved strong predictive performance and can help identify factors influencing student success.
